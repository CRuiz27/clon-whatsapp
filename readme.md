# Proyecto base para la comision 999200928
## Maquetación de proyecto web clon de WhatsApp

Aqui se encuenta la base de mi poyecto de maquetación web

-En la primera entrega de este proyecto se puede apreciar la visualizacion de una interfaz tipo WhatsApp, la cual contiene un sidebar y un ejemplo de un chat.

-El proyecto fue realizado con HTML y CSS, sus archivos principales son:

/index.html  
/styles.css  
/assets/images

-Para su funcionamiento debera abrir el archivo index.html en cualquier navegador.

-Se utilizo un picker de color para aproximar los colores lo maximo posible a el ejemplo, tambien se utilizo la fuente Roboto que es la utilizada por WhatsApp y sus estilos fueron organizados de arriba hacia abajo y de izquierda a derecha.

-El repositorio contiene commits creados durante el desarrollo para mejorar su entendimiento.
